##
# Text Search

##### _An exercise in BDD Development._

#### Mike Chastain

## Description

_Enter a word into one input form.  It returns a results page with, not only the number of times the word exists in the long string to search in.  It also formats the input string to be highlighted in the body of the returned text_

## Setup

Clone from github. then...

```
$ composer install
```

_then start up a local PHP server from within the "web" directory within the project's folder and point your browser to whatever local host server you have created._  

## Technologies Used

_This project makes use of PHP, the testing framework [PHPUnit](https://phpunit.de/), the micro-framework [Silex](http://silex.sensiolabs.org/), and uses [Twig](http://twig.sensiolabs.org/) templates._
