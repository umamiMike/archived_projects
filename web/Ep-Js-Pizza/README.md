# _A small pizza shop app which will undoubtedly blow your friggin mind_
## _A user orders a pizza...what happens next...you decide.  The only thing which is definitely true is you will NOT recieve a pizza at the end of it._
### By Mike Chastain
## Description
_Clone into local repo and open index.html in the browser_

## Technologies Used
_This project makes use of: [Mocha](https://mochajs.org/), and [Chai](http://chaijs.com/) as testing-frameworks. And has a dependency for jQuery._
